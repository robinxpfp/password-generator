# Password Generator

A simple web application to generate random passwords.

![Screenshot](./screen.png)

## Features

- Generates random passwords with various characters.
- Easy-to-use user interface.
- Protect your information with strong passwords.

## Technologies Used

- HTML
- CSS (Tailwind CSS)
- JavaScript
